# 基于物理信息神经网络（PIML）的氧化锆材料逆向设计

## 1. 实验目的
本实验旨在利用结合了物理定律（阿伦尼乌斯定律）的机器学习模型，通过遗传算法在多维空间中反向搜索，寻找在 800°C 下具有最高离子电导率的氧化锆（ZrO2）共掺杂配方，并验证晶格应变理论在材料筛选中的有效性。

## 2. 实验方法

### 2.1 数据处理与特征工程
* **数据来源**：从 DuckDB 数据库提取实验样本，包含合成方法、掺杂元素及其物理属性。
* **特征计算**：计算加权平均离子半径、平均价态等物理特征，并对"合成方法"等分类特征进行 One-Hot 编码。

### 2.2 模型架构 (PIML)
* 采用 **Physics-Informed Neural Network (PIML)** 架构。
* **编码器**：使用多层感知机（MLP）提取材料特征。
* **物理约束**：网络输出层包含物理参数头（活化能 $E_a$ 和 指前因子 $A$），并通过嵌入 **阿伦尼乌斯方程 (Arrhenius Law)** 直接约束预测结果，确保预测的电导率符合物理规律。

### 2.3 逆向设计策略
* 使用 **遗传算法 (Genetic Algorithm, GA)** 进行优化。
* 模拟自然选择过程（交叉、变异），针对掺杂元素的种类（Sc, Mg, Y 等）和摩尔比例进行迭代搜索，目标函数为最大化 800°C 下的电导率。

---

## 3. 实验结果

### 3.1 优化过程分析
遗传算法从初始的随机搜索快速收敛至最优解。
* **第 0 代**：最佳 Log 电导率为 -1.228。
* **第 5 代**：优化至 -1.058 并基本达到平台期，表明算法已快速锁定最优区域。后续迭代中仍有微小提升，最终在第 24 代收敛至 -1.036。

### 3.2 最佳配方发现 (AI-Discovery)
实验最终锁定了一个 **三元共掺杂体系 (Ternary Co-Doping)**。

| 参数 | 数值 | 说明 |
| :--- | :--- | :--- |
| **材料体系** | **ZrO2 - Sc - Mg** | 钪镁共掺杂 |
| **主要掺杂剂** | **Sc (Scandium)** | 摩尔分数: 7.50% |
| **辅助掺杂剂** | **Mg (Magnesium)** | 摩尔分数: 3.19% |
| **总掺杂量** | **10.68%** | - |
| **有效阳离子半径** | **87.60 pm** | 精细调节以匹配晶格 |
| **预测电导率 (800°C)** | **0.092 S/cm** | Log10: -1.036 |
| **推荐烧结温度** | **1505°C** | 相比原方案有所提升 |

### 3.3 物理理论验证
对模型预测的活化能 ($E_a$) 与平均掺杂半径进行了抛物线拟合，以验证 **晶格应变理论 (Lattice Strain Theory)**。拟合得到的理论最优半径为 43.97 pm，落在实验数据范围（~60-112 pm）之外，说明在当前数据分布下，活化能随半径的增大呈现整体下降趋势，抛物线的上升分支尚未被充分采样。尽管如此，实验最佳点（Sc-Mg 组合，~87.60 pm）确实位于活化能较低的区域，表明模型捕捉到了通过调整离子半径来降低离子迁移势垒的物理趋势。

## 4. 结论与建议
1.  **AI 发现成果**：模型成功识别出 **Sc (7.5%) + Mg (3.2%)** 的共掺杂组合。模型解释认为，该组合产生的平均阳离子半径利用"熵稳定效应"最小化了活化能。
2.  **下一步计划**：建议实验人员根据上述配方，在 **1505°C** 条件下制备样品，并进行电化学阻抗谱 (EIS) 测试以验证预测的电导率。
