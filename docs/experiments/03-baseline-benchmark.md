# 基于 PIML 与基准模型的材料电导率预测对比

## 1. 实验概要
**实验名称**：实验 3 - 基准模型训练与多模型综合评估
**实验日期**：2026/02/11
**实验目的**：
1. 构建并训练一个纯数据驱动的深度神经网络 (Standard DNN) 作为强基准 (Baseline)。
2. 将本课题提出的物理信息神经网络 (PIML) 与基准模型、传统机器学习模型 (Random Forest, XGBoost) 在氧化锆材料电导率预测任务上进行性能对比。
3. 验证模型在虚拟筛选新材料 (Virtual Screening) 中的应用潜力。

## 2. 实验设置与方法

### 2.1 数据流与特征工程
* **数据来源**：通过 `MaterialDataProcessor` 从 DuckDB 数据库中提取，包含掺杂剂属性（半径、价态、摩尔分数）、烧结工艺参数及文本描述等。
* **特征预处理**：
    * **数值特征**：使用 `StandardScaler` 进行标准化。
    * **分类特征**：使用 `OneHotEncoder` 编码。
    * **文本特征**：使用 TF-IDF + SVD 降维处理 `material_source_and_purity` 字段。
* **数据集划分**：训练集/测试集比例为 8:2，随机种子 (Seed) 设为 42。

### 2.2 模型架构对比
* **Baseline DNN (基准深度网络)**：
    * **架构**：包含一个 3 层 MLP 编码器 (128-64-32 神经元) 和一个回归头。
    * **输入处理**：温度被标准化 (Z-Score) 后直接作为特征输入网络，不包含任何物理公式约束，代表纯数据驱动的"黑盒"方法。
* **对比模型**：Random Forest (随机森林)、XGBoost、PIML (物理信息神经网络，本实验核心模型)。

## 3. 实验结果分析

### 3.1 预测性能量化对比
根据实验结果，各模型在测试集上的表现如下表所示：

| 模型 (Model) | RMSE (均方根误差) | R² Score (决定系数) | 评价 |
| :--- | :--- | :--- | :--- |
| **PIML (Ours)** | **0.2532** | **0.9353** | **性能优异**，与纯 DNN 相当，兼具物理可解释性。 |
| **Standard DNN** | 0.2518 | 0.9360 | 表现略优，证明深度学习在此数据集上显著优于传统 ML。 |
| Random Forest | 0.7222 | 0.4740 | 性能较差，无法捕捉复杂的材料-工艺非线性关系。 |
| XGBoost | 0.7179 | 0.4803 | 性能一般，远逊于深度学习模型。 |

**分析结论**：
1. **深度学习优势显著**：PIML 和 Standard DNN 的 $R^2$ 均超过 0.93，而传统集成学习模型仅在 0.47-0.48 之间。这说明深度神经网络能更好地提取复杂的材料成分与电导率之间的映射关系。
2. **PIML 的有效性**：PIML 模型在引入物理约束（如 Arrhenius 方程结构）的情况下，依然保持了与纯数据驱动 DNN 高度接近的精度（RMSE 仅高出约 0.0014）。这表明物理信息的引入没有显著牺牲拟合能力，同时赋予了模型泛化性与可解释性。

## 4. 应用验证：虚拟筛选 (Virtual Screening)
利用训练好的 PIML 模型，对不同掺杂元素 (Sc, Y, Yb, Gd) 和浓度 (0.06-0.16) 的虚拟材料进行了筛选。在 800°C 下，预测电导率最高的 Top Candidates 均为 **Sc (钪) 掺杂** 体系。

**Top 5 推荐材料列表**：

| 排名 | 样本 ID | 掺杂元素 | 掺杂浓度 | 预测活化能 ($E_a$, eV) | 预测电导率 ($\sigma$, S/cm) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | Virtual_Sc_0.16 | Sc | 0.16 | 0.9926 | 0.0598 |
| 2 | Virtual_Sc_0.14 | Sc | 0.14 | 0.9924 | 0.0596 |
| 3 | Virtual_Sc_0.12 | Sc | 0.12 | 0.9923 | 0.0594 |
| 4 | Virtual_Sc_0.10 | Sc | 0.10 | 0.9921 | 0.0592 |
| 5 | Virtual_Sc_0.08 | Sc | 0.08 | 0.9920 | 0.0590 |

**筛选发现**：
* 模型一致性地预测 **Sc (钪)** 掺杂在当前工艺条件下能获得最高的电导率（约 0.06 S/cm）。
* 预测的活化能 ($E_a$) 约为 0.99 eV，且随浓度变化保持稳定。

## 5. 结论
本实验成功建立了一个高性能的深度学习基准模型，证明了在该材料数据集上，**深度学习方法显著优于传统机器学习方法** (RMSE 降低明显)。此外，**PIML 模型**在保持高精度的同时 (RMSE: 0.2532)，具备了物理可解释性，并通过虚拟筛选成功推荐了以 Sc 为掺杂剂的潜在材料，达成了实验预期目标。
